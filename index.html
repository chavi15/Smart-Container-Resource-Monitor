How to connect your pipeline
JENKINS Add to Jenkinsfile
post { always { sh "curl -X POST http://localhost:3000/api/cicd -H Content-Type:application/json -d {tool:Jenkins,pipeline:${env.JOB_NAME},status:${currentBuild.result},branch:${env.GIT_BRANCH}}" } }
GITHUB ACTIONS Add workflow step
- name: Notify CRMTR if: always() run: curl -X POST http://localhost:3000/api/cicd -H Content-Type:application/json -d {tool:GitHub Actions,pipeline:${{github.workflow}},status:${{job.status}},branch:${{github.ref_name}}}
